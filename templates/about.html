About UCLex

The UCL Exome (UCLex) Consortium is a coalition of investigators seeking to aggregate and harmonize exome sequencing data from a variety of large-scale sequencing projects, and to make summary data available for the wider scientific community.

The data set provided on this website spans 4539 individuals sequenced as part of various disease-specific and population genetic studies. All of the raw data from these projects have been reprocessed through the same pipeline, and jointly variant-called to increase consistency across projects.

A list of the UCLex Principal Investigators and groups that have contributed data to the current release is available below.

For bug reports, please file an issue on Github.
